機器視覺與自動化工程的「閉環算力中心」
光學計算機 V5 Professional 是一款專為 AOI 視覺工程師、光學架構師與自動化開發人員打造的專業級互動式評估工具。它打破了傳統光學工具單向計算的限制，將感測器晶片庫、幾何光路、多相機拼接、動態景深模擬、光源反射夾角以及編碼器多倍頻解碼深度整合於單一輕量化架構中，為工業現場調校與前期規格敲定提供秒級的精準推導。

五大核心技術亮點
求解目標定向閉環引擎 (Closed-Loop Engine)
擺脫「輸入 A 才能算 B」的傳統邏輯。使用者可自由將 FOV、工作距離 (WD) 或鏡頭焦距設定為「最終求解目標」，系統將動態解鎖欄位、即時閉環反推，絕不產生數據斷層與 NaN 錯誤。

多相機拼接拓寬運算 (Multi-Camera Stitching Analyzer)
針對大型輸送帶（如鋰電、鋼板檢測）橫向寬幅不足的痛點，導入「相機重疊率 (Overlap %)」動態估算模型，代入單台光學規格即可自動反推所需的最少安裝台數與實際物理總涵蓋寬度。

智慧打光幾何分類器 (Lighting Geometry Analyzer)
結合 Tab 4 互動式光路藍圖，輸入光源外徑與高度即可自動精確計算邊緣光束入射夾角（精準至 0.1°），並智慧判定當前打光型態屬於「高角度正向光 (Brightfield)」、「低角度側向光 (Darkfield)」或「穿透式背光 (Backlight)」，完美評估金屬劃痕與輪廓提取。

A/B 相多倍頻硬體解碼同步 (Encoder Quadrature Mode)
直覺對接現場工業採集卡架構。支援 1x、2x、4x 脈衝倍頻選單，自動細分物理間距，將機械輪軸的實際線速度與相機極限掃描頻率 (Line Rate) 無縫綁定。

PWA 邊緣端廠房離線部署 (Edge Offline Deployment)
內建標準 PWA 漸進式 Web 應用架構。無需連網、不占記憶體，一鍵導出標準設定檔後，即可在半導體無塵室、極限機台現場等斷網環境下直接進行本地離線安裝與極速啟動。

🎯 適用場景
前期專案評估：快速確認相機像素、鏡頭焦距與工作距離的幾何可行性。

現場機台調校：精確計算編碼器硬體脈衝間距與曝光極限時間，防止影像拉伸與動態模糊。

光學打光設計：透過視覺化幾何光束軌跡，直觀判斷劃痕、邊緣及表面瑕疵的打光角度。
